Extra Headers setup

Set the names of your extra headers

For compatibility a custom header name should start with an "X-" prefix, for example "X-CoolHeader"

If you set extra header with the same name that may exist in request it will be overridden.
For example you can override User-Agent header by defining your own User-Agent header and setting your own value

Custom Header name should not contain spaces or non-ASCII characters
